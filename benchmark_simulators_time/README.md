# Compare OneSC and BoolODE runtime (single-core)
This folder contains code and data to compare the runtime between OneSC simulator and BoolODE (single-core). This analysis generates **Fig 3A**. 

To run this, please first download and extract [BoolODE](https://cnobjects.s3.amazonaws.com/OneSC/manuscript_github_data/benchmark_simulators_time/BoolODE.tar.gz) folders into this directory. Then either create or download the [output](https://cnobjects.s3.amazonaws.com/OneSC/manuscript_github_data/benchmark_simulators_time/output.tar.gz) folder. 

To run the analysis, run the scripts in *scripts/run_main.sh*, *scripts/run_main_2.sh* and *scripts/run_main_3.sh*.
